Clear Clear download history

Clear browser download history when a PDF file is downloaded.
Path Enter Path to a PDF viewer

Note: for example the path to Adobe PDF Reader in Windows 10 is:

C:\Program Files (x86)\Adobe\Acrobat Reader DC\Reader\AcroRd32.exe
Arguments Add arguments for PDF viewer

Note: a sample argument in Adobe Acrobat Reader is:

zoom=1000

Note: please use - & - operator if you have more than one argument:

zoom=1000&toolbar=0
